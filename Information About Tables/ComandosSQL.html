-------------STRUCTURE QUERY LANGUAGE------------- SELECT * FROM `veiculos` - Vai selecionar tudo(*) da tabela veiculos. SELECT fabricante, modelo FROM `veiculos` - Vai selecionar as colunas do fabricante e o modelo da tabela veiculos. SELECT * FROM `veiculos` WHERE fabricante = "citroen" - Vai chamar apenas os veiculos onde o fabricante é citroen. SELECT * FROM `veiculos` WHERE fabricante = "citroen" LIMIT 10 - Vai chamar apenas os veiculos onde o fabricante é citroen e irá buscar apenas 10 veículos.
